Qu'est-ce que HTML, concrètement ?

HyperText Markup Language (HTML) est le langage utilisé pour organiser une page web et ce qu'elle peut contenir.

Personne ne programme avec le HTML parce que c'est ne pas un langage de programmation, Je dirais plutôt que les développeurs tapent du code HTML sous forme des balises pour définir qui va jouer le rôle d'un paragraphe, d'une image, des tableaux, ... C'est un langage de balises.

Qu'est-ce que CSS, pratiquement ?

Le Cascading Style Sheets(CSS) sont le code utilisé pour mettre en forme une page web. Si tu vois le HTML comme le dessin du plan de ta maison, voit le CSS comme les crayons de couleurs avec lesquelles tu vas donner une forme visuelle un peu réaliste à ton plan. De la même façon que HTML, Personne ne programme avec le CSS parce que c'est ne pas un langage de programmation, Je dirais plutôt que les développeurs tapent du code CSS sous forme des balises pour définir qui va jouer le rôle d'un paragraphe, d'une image, des tableaux, ... C'est un langage de feuille de style

Qu'est-ce que le JavaScript, réellement ?

JavaScript (JS en sigle) est un langage de script léger à ne pas confondre avec le langage Java d'Oracle. Il est orienté objet, principalement connu comme le langage de script des pages web parce qu'il est interprété par presque tout le navigateur. Par défaut, tu as interpréteur ou un moteur de script JavaScript Intégré dans ton navigateur. Mais tu vas aussi le trouver dans d'autres environnements extérieurs aux navigateurs web tels que Node.js, Apache CouchDB , MongoDb, ... Le code JavaScript est interprété ou compilé à la volée. C'est un langage à objets qui permettent de programmer suivant plusieurs paradigmes(vision de choses) de programmation rencontré.
